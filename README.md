# MangaLib Desktop

Десктопная обёртка для сайта [mangalib.me](https://mangalib.me) на Python и PyWebView.  
Позволяет читать мангу, манхву, ранобэ и смотреть аниме (через F11 можно будет переключиться на AnimeLib) в отдельном окне — без вкладок браузера, с сохранением входа и удобным переключением режимов.

## Возможности

- Полноэкранный и оконный режимы с переключением по `F11`
- Меню с выбором языка (RU / EN) и кнопкой смены режима
- Сохранение данных сайта (логин, настройки) между запусками
- Иконка приложения (белая `M` на чёрном фоне с оранжевыми линиями, как и у самого сайта. Можете создать ярлык для exe-файла и добавить себе на рабочий стол)
- Лёгкий, без лишних зависимостей

## Установка и запуск

1. Скачай последний релиз `MangaLib.exe` со страницы [Releases](https://github.com/Doroko-Dorokovich/MangaLib-Desktop/releases)
2. Запусти файл — приложение откроется на весь экран (развёрнутое окно)
3. Нажми `F11`, чтобы открыть меню и переключиться в полноэкранный режим или обратно
